# emacs.d settings

## Description

My very own emacs.d configuration with everything I like.

## Setup

To grab all the dependencies:

    git clone git@github.com:aminRX/emacs.d.git

## Install all-the-icons

Run in terminal:

```sh
$ emacs --insecure
```

Run in emacs:

    git clone git@github.com:aminRX/emacs.d.git
