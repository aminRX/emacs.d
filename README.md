# emacs.d settings

## Description

My very own emacs.d configuration with everything I like.

## Setup

Quick Install:

    git clone https://github.com/aminRX/emacs.d ~/.emacs.d

## Install all-the-icons

Run in terminal:

```sh
$ emacs --insecure
```

Run in emacs:

    M-x all-the-icons-install-fonts
